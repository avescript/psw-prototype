# Hello World Profile Page

A simple profile page with a circular photo and contact button. Built with pure HTML and CSS.

## Features
- Responsive design
- Circular profile photo
- Contact button with hover effect
- Clean, modern UI with a blue theme

## Setup
Simply open `hello.html` in a web browser to view the page. No additional dependencies required.
